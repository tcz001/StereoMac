# StereoMac

A stereo vision Screen capture tool for Mac OS X

Using:

- OpenCV
- Quartz Display Service for faster in memory screen capture
- Barrel Distortion for stereo vision calibration

## Quick Start

```
cmake . && make && ./DisplayStereo
```
